# Criando Máquinas Virtuais na Azure

## Descrição do Projeto

Este repositório foi criado como parte do desafio da DIO com o objetivo de documentar os conhecimentos adquiridos sobre criação e gerenciamento de máquinas virtuais na Microsoft Azure.

## Objetivos do Lab

Durante este laboratório, foram abordados conceitos importantes relacionados à computação em nuvem e à criação de máquinas virtuais utilizando o portal Microsoft Azure.

Os principais objetivos foram:

* Compreender o funcionamento das máquinas virtuais na Azure;
* Aprender a criar e configurar uma VM;
* Conhecer recursos de rede, armazenamento e segurança;
* Utilizar o portal Azure para gerenciamento de serviços;
* Documentar o processo utilizando GitHub e Markdown.

## O que aprendi

### Introdução às Máquinas Virtuais

Aprendi que as máquinas virtuais (VMs) permitem executar sistemas operacionais e aplicações na nuvem sem a necessidade de hardware físico local.

A Azure oferece diferentes tipos de máquinas virtuais para diversos cenários, como:

* Desenvolvimento;
* Testes;
* Hospedagem de aplicações;
* Banco de dados;
* Ambientes corporativos.

### Processo de Criação da VM

Durante o laboratório, aprendi o passo a passo para criar uma máquina virtual no portal Azure:

1. Criar um grupo de recursos;
2. Definir nome da máquina virtual;
3. Escolher região e sistema operacional;
4. Selecionar tamanho da VM;
5. Configurar usuário e senha;
6. Configurar portas de acesso;
7. Revisar e criar a máquina virtual.

### Recursos Importantes

Também aprendi sobre:

* Endereço IP público;
* Disco de armazenamento;
* Regras de firewall;
* Rede virtual (VNet);
* Segurança e monitoramento.

## Benefícios da Computação em Nuvem

O laboratório também mostrou vantagens importantes da computação em nuvem:

* Escalabilidade;
* Alta disponibilidade;
* Redução de custos;
* Flexibilidade;
* Facilidade de gerenciamento.

## Conclusão

Este desafio foi importante para consolidar os conhecimentos sobre Microsoft Azure e máquinas virtuais. A experiência prática ajudou a entender melhor como funciona a criação e administração de recursos em nuvem utilizando o portal Azure.
